## How to run locally

```bash
# 1. Create virtual environment
python -m venv env

# 2. Activate virtual environment
env\Scripts\activate.bat          # for Windows
    # or
source env/bin/activate           # for Linux

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run server
python app.py
```
